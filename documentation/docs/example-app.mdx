## Building the App

```bash
# clone the project
git clone https://github.com/popile-media/react-native-snapchat-camera-kit.git

# go into the project
cd react-native-snapchat-camera-kit

# make project ready
npm run bootstrap

# go into the example
cd example

# copy environment file and set your credentials
cp .env.example .env

# run for android
npm run android

# or

# run for ios
npm run ios
```
